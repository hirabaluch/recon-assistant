# Recon Assistant 🔍

A simple CLI-based reconnaissance tool for subdomain and endpoint discovery.

## Features
- Subdomain scanning
- Endpoint discovery
- Status code detection
- Saves results to file

## Installation

```bash
git clone https://github.com/hirabaluch/recon-assistant.git
cd recon-assistant
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
